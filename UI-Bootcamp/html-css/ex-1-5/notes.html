CSS Grid Layout

My approach:

To create a header, two sidebars and the main content, first I have set the layout of my display to grid.

To arrange the content, I have used grid-template-areas so that I can differentiate between header, sidebars and the main content.

Used grid-template-columns and grid-template-rows to arrange them by applying minmax() and providing certain sizing to each of them.

Key Features Used

  • grid-template-areas: Defined a structured layout with named areas for header, sidebars, and main content.
  • minmax(): Used for rows and columns to create flexible and responsive sizing.
  • Media Queries: Adjusted grid layout for mobile screens by stacking areas vertically.

Strategies used

  • Ensured consistent layout scaling using fractional units (fr).
  • Handled overflow in the main content area with overflow: auto;.

Challenges

  • Balancing content alignment and spacing in narrow viewports.
