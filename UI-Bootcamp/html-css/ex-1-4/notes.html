CSS Units and Responsive Design

My approach:

After creating a text block to align it to the centre of the viewport, I have used vh and vw.

To set flexible values for the CSS properties like height, width, font-size and margins, I have used the functions like min(), max(), clamp().

For sizing the units like padding, fonts and margins, I have used %,em and rem.

Key Units Used

  • vh and vw: Used for centering and creating layouts that adapt to the viewport.
  • %, em, rem: Used for flexible spacing and typography for relative sizing.
  • min(), max(), and clamp(): Utilized for dynamic font sizes and widths for scalability.

Responsive Strategies

  • Flexible Grids: Designed layouts using grid-template-columns with flexible fractions.
  • Fluid Typography: Implemented text sizing with clamp() to adapt across screen sizes.
